## Features

-  Search Employees by name  
-  Add New Employee with profile image upload  
-  Update Employee details  
-  Delete Employee with confirmation  
-  View Employee Details via dynamic routing  
-  Pagination for scalable data display  
-  Toast Notifications for user feedback  
-  Responsive Design across devices

## How It Works
- Frontend makes API calls to backend using Axios.
- Backend handles routes for CRUD operations and connects to MongoDB.
- Toastify provides instant feedback on actions.
- React Router enables dynamic navigation to employee detail pages.

## Installation 

- Clone the repo
 ```
- git clone https://github.com/NoorAisha25/Employee-Management-App.git
```
- Install backend dependencies
```
cd backend
npm install
```
- Install frontend dependencies
```
cd ../frontend
npm install
```
- Start backend
```
npm run server
```
- Start frontend
```
npm start
```
